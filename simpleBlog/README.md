# Simple blog
Full fetature blog with create,delete posts and comments.

## Features
- Create posts and delete it
- Comment on posts and delete it
- login page and safe user data

## Tech Stack
- Frontend: Bootstrap
- Backend: Flask, Python
- Database: MySQL
- Authentication: Flask-Login
